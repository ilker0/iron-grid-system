## ðŸ”¨ Usage

#### For Vue 2.x

```
import Vue from "vue";
import App from "./App.vue";
import 'iron-grid-system/dist/style.css';

new Vue({
  render: h => h(App)
}).$mount("#app");

```

#### For Vue 3.x

```
import { createApp } from "vue";
import App from "./App.vue";
import 'iron-grid-system/dist/style.css';

const app = createApp(App);
app.mount("#app");

```
